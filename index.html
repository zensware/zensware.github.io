Documentation

Simple, step-by-step setup for Windows users. No technical background required.

1) Quick start (EXE)
  1. Download the zip above and extract it.
  2. Right-click the .exe and choose Run as administrator.
    Admin mode lets global hotkeys work reliably.
  3. On first run an on-screen status box (OSD) appears at the top-left.
  4. If Windows SmartScreen shows a warning, choose More info → Run anyway.
2) First run basics
  • OSD: shows On/Off, current Mode, and the active Shoot Bind.
  • Two modes:
    • HOLD: active only while you hold the chosen key or mouse button.
    • TOGGLE: press once to turn on, press again to turn off.
  • Quit at any time with the Delete key.
3) Controls you need
  • Toggle on/off in toggle mode: Insert.
  • Hold mode: press and hold your HOLD_KEY (default mouse4).
  • Switch the Shoot Bind: press Right Arrow to cycle through options. Match the Alternate Shootbind in your other software to what the OSD shows. If the key stops working, press Right Arrow again until it matches.
  • Quit: Delete.
4) Config file (optional)

After the first run, a config.json is created next to the executable. You can edit it with Notepad.

Keys explained
  • MODE: "hold" or "toggle".
  • HOLD_KEY, TOGGLE_KEY, QUIT_KEY: hotkeys you prefer.
  • SHOOT_BIND_OPTIONS: list of keys to send on detection.
  • SHOOT_BIND: which one of the above is active on start.
  • SCAN_SIZE: scan square half-width in pixels around the screen center.
  • TOLERANCE: color match tolerance. Larger = more matches.
  • CLICK_DELAY: seconds between key presses; 0 uses a fast default.
Template 
{
  "MODE": "hold",
  "SHOOT_BIND": "8",
  "SHOOT_BIND_OPTIONS": ["7","8","9","0"],
  "HOLD_KEY": "mouse4",
  "TOGGLE_KEY": "insert",
  "QUIT_KEY": "delete",
  "SCAN_SIZE": 2,
  "TOLERANCE": 10,
  "CLICK_DELAY": 0
}
5) Common issues
  • Hotkeys do nothing: re-run the app as Administrator.
  • OSD not visible: check that it isn't behind overlays; ensure transparency effects are on in Windows Settings.
  • High CPU: lower SCAN_SIZE or increase CLICK_DELAY in config.json.
  • It never triggers: increase TOLERANCE a little or check that the active Shoot Bind matches your other software's input.
  • Wrong key sent: press Right Arrow to cycle binds until the OSD shows the one you expect.
  • Stuck On/Off status: in HOLD mode ensure the correct mouse side button is set (mouse4 vs mouse5).
